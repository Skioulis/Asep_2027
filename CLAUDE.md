## Project Overview

This is a Python project for processing and working with ASEP (Ανώτατο Συμβούλιο Επιλογής Προσωπικού — Greece's Supreme Civil Service Council) exam question banks. The `questions/` directory holds `.docx` source files containing exam questions across multiple subjects (constitutional law, administrative law, economics, informatics, modern Greek history, etc.).

## Running

```bash
python main.py
```

## Key Files

- `main.py` — main entry point (currently a stub)
- `questions/` — source `.docx` files with ASEP exam questions per subject area
